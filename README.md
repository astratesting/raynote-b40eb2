# Raynote

Cross-device syncing notes app for students. Capture lecture notes on any device and access them instantly anywhere — no formatting loss, no sync delays.

## Features

- **Instant Cross-Device Sync** — Start typing on your laptop, continue on your phone
- **Zero Formatting Loss** — Everything renders perfectly on every device
- **Built for Lectures** — Voice-to-text, quick templates, smart organization
- **Offline-First Capture** — Keep writing offline, syncs when you're back online
- **Lightning-Fast Search** — Find any note in milliseconds

## Tech Stack

- Next.js 14
- TypeScript
- Tailwind CSS
- Prism gradient design system (flame-orange, vivid-magenta, acid-green)

## Getting Started

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) to view the landing page.
